# Playbooks

Use this directory for durable, multi-step procedures that agents may need to
follow repeatedly.

Current playbooks:

- `run-code.md`: choose and run the correct F-flat-minor runtime for `.ff` and `.ffp` files.
- `test-and-dev-workflows.md`: common test, lint, and development commands.

Add new playbooks here when a workflow is recurring, ordered, and too procedural
for `AGENTS.md` or `.agents/AGENTS.md`.
